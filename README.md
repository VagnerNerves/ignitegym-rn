## ðŸš€ Running the Project

### Back-end

Clone the project

```bash
  git clone https://github.com/VagnerNerves/ignitegym-rn.git
```

Enter the project directory

```bash
  cd ignitegym-rn\server
```

Install with dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

Access the README.md from the server folder
